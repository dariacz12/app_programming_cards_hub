## ğŸ“¦ Tech Stack

- **Framework:** [Expo SDK 54](https://docs.expo.dev/) + React Native 0.81
- **Navigation:** React Navigation (Bottom Tabs + Native Stack)

- **APIs:** Axios, Expo Secure Store, Notifications
